Model Notes

equation core

Intensity combines a warm central source with a moving light-echo surface intersecting uneven dust sheets.

I = S_star + (E_volume + E_shells) * T_dust * M_asym

z_echo = (r^2 - t^2) / 2t

E_volume = sum w_i exp(-((z_echo - z_sheet_i)^2) / 2sigma_i^2)

RGB = tone(background + warm_dense_dust + cool_thin_scatter)
